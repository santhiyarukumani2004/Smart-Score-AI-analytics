# Smart Score AI 🚀

An AI-powered exam analytics dashboard for teachers and students, built with React and Vite.

## ✨ Features

- 📊 **Tweet Stress Analysis** - Real-time stress patterns from Twitter data
- 😂 **Meme Sarcasm Detection** - Analyze sarcasm and stress in memes  
- 📈 **Difficulty Classification** - Categorize topics by difficulty levels
- 🎯 **Topic Analysis** - Identify hardest and easiest topics
- 📱 **PWA Support** - Install as native app on any device
- 🎨 **Google AI Dashboard Design** - Modern, clean interface
